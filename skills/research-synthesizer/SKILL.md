# Goal
Perform deep research on a user-provided topic and output a professional summary.

# Instructions
1. **Search**: Conduct 3-5 targeted web searches to find authoritative sources.
2. **Synthesize**: Extract the core concepts, current trends, and major players.
3. **Analyze**: Specifically identify "avant-garde" monetization opportunities for an AI agent in this niche.
4. **Format**: Output exactly 5 bullet points for findings and a separate table for monetization strategies.

# Constraints
- Use only verified information from the last 12 months.
- If a specific tool or API cost is found, include it in the report.
- Do not exceed 500 words total.

# Success Criteria
- The summary must be actionable for a business task.
- At least one strategy must focus on automated agent-to-agent transactions.
